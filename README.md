## Overview
This module replaces the stock KernelSU app with a modified version that uses better approach for minimizing sparse image.

## The 2TB Storage Bug Fix
KernelSU v1.0.1 contains a known 2TB storage bug. While the official app provides a "Minimize sparse image" workaround in the settings, it frequently causes module compatibility issues. 

This module directly integrates the **KernelSU Evo** solution to properly resolve the 2TB bug at its core. This ensures all modules work perfectly without compromises, making it an essential tool for non-GKI kernel developers and users who still rely on KernelSU v1.0.1.

## Installation & Usage
1. Install the [MMRL app](https://github.com/MMRLApp/MMRL/releases/latest/download/normal.apk) and grant it root permissions via KernelSU.
2. Install this module through MMRL.
3. After installing your desired modules, go to the app's settings and press **Fix 2TB storage**.
