## Overview
This module replaces the stock KernelSU app with a modified version that uses a better approach for minimizing the sparse image.

## The 2TB Storage Bug Fix
KernelSU v1.0.1 contains a known 2TB storage bug. While the official app provides a "Minimize sparse image" workaround in the settings, it can cause module compatibility issues. 

This module integrates the **KernelSU Evo** solution to address this bug, improving module compatibility for non-GKI kernel developers and users relying on KernelSU v1.0.1.

## Installation & Usage
1. Install the [MMRL app](https://github.com/MMRLApp/MMRL/releases/download/v34296/MMRL-v34296-release-official.apk) and grant it root permissions via KernelSU.
2. Install this module through MMRL.
3. After installing your desired modules, go to the app's settings and press **Fix 2TB storage**.
